= VCL-total FFI Integration Tests

`integration_test.zig` exercises the full query pipeline through the exported C
functions, verifying ABI version, lifecycle, error handling, safety level
progression across all three query modes, memory safety (double-free, use-after-free),
and QueryPlanHeader struct layout against the Idris2 Layout.idr definitions.
