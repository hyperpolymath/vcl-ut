= VQL-UT FFI Source

`main.zig` implements the complete VQL-UT FFI: a thread-safe slot pool of query
contexts, the 5-stage query pipeline (parse, bind_schema, check_types,
check_effects, compile), safety level inspection, and handle lifecycle management.
