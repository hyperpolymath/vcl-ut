= VQL-UT FFI Implementation (Zig)

Zig implementation of the C-compatible FFI layer for VQL-UT.
Uses a thread-safe, slot-based query context pool for concurrent operation.

== Building

    cd src/interface/ffi && zig build test

== Subdirectories

`src/`:: Main FFI source (main.zig) implementing all exported C functions.
`test/`:: Integration tests verifying FFI against the Idris2 ABI contract.
