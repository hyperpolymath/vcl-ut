= VCL-total Source Root

This directory contains the VCL-total (VCL Total Type-Safety) source tree.
VCL-total is a 10-level progressive query safety checker for VeriSimDB backends.

== Subdirectories

`interface/`:: ABI definitions (Idris2), FFI implementation (Zig), and generated C headers.
`aspects/`:: Cross-cutting concerns: integrity, observability, and security aspects.
`contracts/`:: Interface contracts and protocol definitions for VCL-total consumers.
`definitions/`:: Core type definitions and domain vocabulary shared across all modules.
`errors/`:: Structured error types for all 10 safety levels plus internal errors.
